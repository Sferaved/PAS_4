## Готовий файл для здачі

**`RGR_variant_1.docx`** — оформлений документ (Times New Roman 14, поля за методичкою, титул, зміст, вступ, теорія, розрахунки, 3 графіки, висновки, список літератури).

## Як змінити варіант і ПІБ

Відкрийте `generate_rgr.py`, на початку файлу змініть:

```python
VARIANT = 1          # ваш номер за журналом (1–10)
STUDENT_NAME = "..."
GROUP = "..."
REVIEWER = "..."
```

Потім у терміналі:

```bash
cd docs\RGR_OOP_2025
python generate_rgr.py
```

Буде створено `RGR_variant_N.docx` та оновлені PNG-графіки.
